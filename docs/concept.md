# CoffeeBlog — 企画・コンセプト立案書

**作成日**: 2026-06-13

---

## 1. ブログコンセプト

### プラットフォーム

**note.com** に継続投稿。このリポジトリは AI 記事生成の自動化インフラ。

### テーマ

日常のコーヒー体験（豆・抽出・カフェ・文化）を AI の力で継続的に発信するコーヒー専門ブログ。

### ターゲット読者

- コーヒーに興味があるすべての人（初心者〜中級者）
- 自宅コーヒーをもっと楽しみたい人

---

## 2. コンテンツカテゴリ

| カテゴリ         | 内容                                           |
| ---------------- | ---------------------------------------------- |
| **豆・産地**     | スペシャルティコーヒー、産地別特徴、焙煎度解説 |
| **抽出・レシピ** | ドリップ、エスプレッソ、コールドブリュー       |
| **カフェ情報**   | 国内外のカフェレビュー・トレンド               |
| **道具・機器**   | コーヒーグッズレビュー・比較                   |
| **コーヒー文化** | 世界のコーヒー文化、ニュース、トレンド         |

---

## 3. AI 活用方針

```
[外部情報収集 / トピック指定]
  ↓
[Claude API で記事下書き生成（Markdown）]
  ↓
[人間がレビュー・加筆]
  ↓
[note.com に投稿]
```

- 投稿頻度: 週2〜4本
- 文体: note 既存記事のスタイルに合わせたプロンプト設計

---

## 4. 技術構成

| 要素           | 採用技術                           |
| -------------- | ---------------------------------- |
| メインブログ   | **note.com**                       |
| AI 生成        | **GitHub Models (GPT-4o)**（無料・追加APIキー不要） |
| スクリプト     | **TypeScript** (Node.js)           |
| 自動化         | **GitHub Actions**                 |
| バージョン管理 | **GitHub**                         |
