## 1. ブログコンセプト

### プラットフォーム

**note.com** に継続投稿。このリポジトリは AI 記事生成の自動化インフラ。

### ビジョン

> 「今日のコーヒーで、世界のどこかへ旅に出る。」

### テーマ

神奈川に暮らしながら、一杯のコーヒーを通じて世界の産地を旅し、日本の最新コーヒーシーンを追いかけ、
地元神奈川の素敵なカフェへ出かけたくなる——そんな**わくわくどきどき**を毎日届けるコーヒー専門ブログ。

### ターゲット読者

- **主要ターゲット**: 神奈川在住（横浜・鎌倉・湘南・川崎・小田原・箱根）の20〜40代
- コーヒー初心者〜中級者
- 世界のコーヒー産地のストーリーが知りたい
- 日本の最新カフェ・トレンドを追いたい
- 週末に地元（神奈川）の素敵なカフェへ出かけたい

## 2. コンテンツカテゴリ（6本柱）

| カテゴリ | 内容 | 月の目標比率 |
| -------- | ---- | ----------- |
| **世界のコーヒー探訪** | 世界各地の産地・農家・ストーリーを旅行記スタイルで | 25% |
| **日本最新コーヒーシーン** | JBC・新規オープン・トレンド・イベント情報 | 20% |
| **神奈川コーヒーマップ** | 横浜・鎌倉・湘南・箱根のローカルカフェ情報 | 15% |
| **豆・産地 深掘り** | 品種・精製・風味の知識記事 | 15% |
| **抽出・レシピ** | 自宅再現できるレシピ、技術解説 | 15% |
| **道具・文化** | 器具レビュー・コーヒー文化読み物 | 10% |

## 3. AI 活用方針

```
[外部情報収集 / トピック指定]
  ↓
[Claude API で記事下書き生成（Markdown）]
  ↓
[人間がレビュー・加筆]
  ↓
[note.com に投稿]
```

- 投稿頻度: 週2〜4本
- 文体: note 既存記事のスタイルに合わせたプロンプト設計
