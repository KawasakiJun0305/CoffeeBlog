# CoffeeBlog コンテンツポリシー

> 最終更新: 2026-06-13 | 管理: CoffeeBlog プロジェクト

note.com に公開するすべてのコーヒー記事に適用されるポリシー。
AI生成・人間執筆を問わず、**公開前に必ずすべてのゲートを通過すること**。

---

## 記事公開フロー（必須）

```
記事生成
  │
  ▼
[Gate 1] リーガルチェック（/legal-check-coffee-article）  ← 必須・スキップ禁止
  │ スコア 75 未満 → 修正して再チェック
  ▼
[Gate 2] 品質レビュー（/review-coffee-article）           ← 必須
  │ スコア 70 未満 → 修正または再生成
  ▼
[Gate 3] 人間による最終確認                               ← 必須・AI判断で代替不可
  ▼
note.com 下書き保存 → 公開
```

**Gate 1（リーガルチェック）は Gate 2 より前に実行すること。**
法的リスクのある記事に品質改善コストをかけることを防ぐため。

---

## Gate 1: リーガルチェック（必須）

### 根拠

AI 生成記事は以下のリスクを内在する:
- 根拠のない健康効果の標榜（薬機法・健康増進法違反リスク）
- 店舗・価格情報の誤記（景品表示法・信用毀損リスク）
- 他記事・公式情報の無断引用（著作権法違反リスク）
- PR未開示（2023年ステルスマーケティング規制違反リスク）

### 実行ルール

| 条件 | ルール |
|------|--------|
| 全記事 | 公開前に `/legal-check-coffee-article` を実行 |
| 🔴 高リスク検出時 | 必ず修正してから再チェック。スキップ禁止 |
| 🟡 中リスク検出時 | 修正を強く推奨。判断が難しい場合は人間が確認 |
| 公開直前 | `--strict` モードで再チェックを推奨 |
| カフェ情報・道具・機器カテゴリ | ステマ規制（Check 2）を特に注意 |
| 健康・栄養に触れる記事 | Check 1 のスコアが 18/20 未満なら公開禁止 |

### スコア基準

| スコア | 対応 |
|--------|------|
| 90〜100 | 法的リスクなし → 公開可 |
| 75〜89  | 🟡項目を修正後に公開可 |
| 60〜74  | 🔴項目を修正し **再チェック必須** |
| 0〜59   | 公開禁止。修正後に全チェックをやり直す |

## Gate 2: 品質レビュー

`/review-coffee-article` で 8 軸スコアリングを実施。

| スコア | 対応 |
|--------|------|
| 85〜100 | 公開可 |
| 70〜84  | 軽微修正後に公開可 |
| 55〜69  | 構成改善後に再レビュー |
| 0〜54   | 再生成 |

---

## Gate 3: 人間による最終確認チェックリスト

note.com への下書き投稿前に、人間が必ず確認する項目:

```
□ タイトルを確認・微調整（30文字以内か）
□ リーガルチェック結果を確認済み
□ 品質レビュー結果を確認済み
□ 価格・営業時間に「公式サイトでご確認ください」があるか
□ PR・提供品がある場合、冒頭に開示表記があるか
□ アイキャッチ画像を設定（自分で用意した素材か確認）
□ [画像: 〇〇] の位置に画像を挿入
□ ハッシュタグを確認（カテゴリ別推奨タグ）
□ プレビュー文（140字）を確認
□ 下書き保存 → 公開判断
```

---

## 禁止コンテンツ（即時廃棄）

以下を含む記事は修正ではなく**廃棄・再生成**とする:

1. 医薬品的効能の断定（「コーヒーで〇〇が治る」等）
2. 実在する個人・店舗への根拠なき誹謗中傷
3. 他サイトの文章の無断コピー（3文以上の一致）
4. 架空の店舗・農園・人物を実在するように記述

|--------|------|------|

| `/generate-coffee-article` | 記事生成 | 生成時 |

## 免責事項

本ポリシーのリーガルチェックは、一般的な法令上の注意点を AI がスクリーニングするものです。
法的判断が必要なケースや、重大なリスクが疑われる場合は、専門の法律家にご相談ください。
