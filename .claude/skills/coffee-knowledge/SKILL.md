# コーヒー知識 網羅的獲得スキル

CoffeeBlogプロジェクトのための、コーヒーに関する知識を体系的に調査・整理・蓄積するスキル。
Web検索で最新情報を収集し、`knowledge/` ディレクトリに構造化されたナレッジベースを構築する。

ターゲット読者（神奈川在住・わくわくどきどき志向）に合わせ、
**世界のコーヒー産地情報**と**日本の最新コーヒーシーン**を重点的にカバーする。

## 使い方

```
/coffee-knowledge [ドメイン] [--deep] [--update]
```

**引数:**
- `[ドメイン]`: 調査対象（省略時は全ドメインを網羅）
  - `beans` — 豆・品種・産地（コアドメイン）
  - `processing` — 精製・加工プロセス
  - `roasting` — 焙煎理論
  - `brewing` — 抽出・レシピ
  - `sensory` — テイスティング・フレーバー評価
  - `equipment` — 器具・機器
  - `culture` — 歴史・文化・トレンド
  - `science` — コーヒーの化学・科学
  - `world-origins` — 世界産地の深掘り（旅行記素材）【NEW】
  - `japan-scene` — 日本最新コーヒーシーン【NEW】
  - `kanagawa` — 神奈川コーヒーマップ情報【NEW】
- `--deep`: 各ドメインを詳細に掘り下げる（通常の2〜3倍の情報量）
- `--update`: 既存の knowledge ファイルを最新情報で上書き更新

---

## 実行手順

### Step 0: 準備

1. `knowledge/` ディレクトリの存在を確認する（なければ作成）
2. ドメイン引数を解析する。省略時は全7ドメインを順番に処理する
3. `--update` 指定時は既存ファイルを読み込んで差分更新する

---

### Step 1: ドメイン別 知識調査

`web-search` スキルを使い、以下のドメイン別調査計画に従って情報収集する。

#### 🌱 beans（豆・品種・産地）

調査項目:
- コーヒーの2大種（アラビカ / ロブスタ）の特性比較
- 主要品種一覧（ゲイシャ・ティピカ・ブルボン・カトゥーラ・SL28等）
- 産地別フレーバープロファイル（エチオピア・コロンビア・ブラジル・グアテマラ・インドネシア等）
- テロワール（標高・気候・土壌）と風味の関係
- スペシャルティコーヒーの定義とCOEスコア基準

検索クエリ例:
```
コーヒー品種 アラビカ ロブスタ 違い
スペシャルティコーヒー 産地別 フレーバー 特徴
エチオピア コロンビア コーヒー テロワール
```

---

#### 🍒 processing（精製・加工プロセス）

調査項目:
- ウォッシュド / ナチュラル / ハニー / アナエロビック の工程と風味への影響
- 精製方法の地域差（エチオピア式 / コスタリカ式等）
- 発酵プロセスの最新トレンド（CO2マセレーション・ラクティック等）
- 精製度合いとカッピングスコアの相関

---

#### 🔥 roasting（焙煎理論）

調査項目:
- メイラード反応 / カラメル化 / 熱分解の仕組み
- 焙煎度の区分（ライト〜イタリアン）と風味変化
- ロースタープロファイルの読み方（ROR・DTR・クラック温度）
- シングルオリジンvsブレンドの焙煎アプローチ

---

#### ☕ brewing（抽出・レシピ）

調査項目:
- 主要抽出方法の比較（ハンドドリップ・エスプレッソ・フレンチプレス・エアロプレス・水出し等）
- 抽出理論（TDS・収率・抽出時間と溶解物の関係）
- SCAゴールデンカップ基準
- バリスタ競技のレシピトレンド
- コーヒー粉量・湯温・挽き目の最適化ガイドライン

---

#### 👅 sensory（テイスティング・フレーバー評価）

調査項目:
- SCAフレーバーホイールの構造と使い方
- カッピングプロトコル（SCA標準手順）
- アロマ・フレーバー・アフターテイストの評価軸
- 欠点豆の識別（フェノール・発酵臭・リオイ等）
- Q Grader資格の評価基準

---

#### 🛠 equipment（器具・機器）

調査項目:
- ドリッパー形状（円錐・台形・波）と抽出特性の比較
- グラインダー種類（コニカルバー・フラットバー・ブレード）の特性
- エスプレッソマシンの種類（シングル・HX・デュアルボイラー）
- スケール・温度計・TDSメーターの選び方
- 器具メンテナンスの重要ポイント

---

#### 🌍 culture（歴史・文化・トレンド）

調査項目:
- コーヒーの起源（エチオピア〜アラビア半島〜欧州への伝播）
- ウェーブ理論（1st / 2nd / 3rd Wave）
- 日本のコーヒー文化（喫茶店文化・サードウェーブの受容）
- サステナビリティ・フェアトレード・直接取引の現状
- 最新トレンド（ノンカフェイン・コールドブリュー・コーヒーカクテル等）

---

#### 🔬 science（コーヒーの化学・科学）

調査項目:
- コーヒーの主要化合物（カフェイン・クロロゲン酸・有機酸・揮発性香気成分）
- 健康影響の最新エビデンス（心血管・認知機能・抗酸化）
- CO2ガスとガスオフの仕組み（鮮度と保存）
- 水質（硬度・pH・ミネラル組成）と抽出の関係
- デカフェ処理方法の比較（スイス・ウォーター・CO2抽出）

---

#### 🌐 world-origins（世界産地の深掘り — 旅行記素材）【NEW】

「世界のコーヒー探訪」カテゴリ記事の素材収集。産地ごとの旅行記・ストーリー素材を集める。

調査対象産地（年間ローテーション）:
- **アフリカ**: エチオピア（イルガチェフェ・ゲデオ・シダモ・ハラー）/ ケニア（キリニャガ・ニエリ）/ タンザニア（キリマンジャロ）/ ルワンダ / ブルンジ / イエメン（モカ・マタリ）
- **中南米**: コロンビア（ウィラ・ナリーニョ）/ コスタリカ（タラス）/ グアテマラ（アンティグア）/ パナマ（ゲイシャ・ボケテ）/ エルサルバドル / ブラジル（セラード）/ ジャマイカ（ブルーマウンテン）
- **アジア・太平洋**: インドネシア（マンデリン・バリ・スラウェシ）/ ベトナム / タイ / ハワイ（コナ）
- **国産**: 沖縄 / 小笠原

各産地について収集すべき情報:
- 地理情報（国・地域・標高・面積）と気候（雨季・乾季・平均気温）
- 農家・協同組合・著名な生産者の名前とエピソード
- 品種（在来種・改良種）と精製方法の特徴
- 風味プロファイル（地域ごとの差異）
- 現地の人々の暮らし・コーヒーとの関わり
- 生産地の歴史的背景（コーヒーがいつ・どうやって持ち込まれたか）
- 日本・神奈川でその産地のコーヒーが手に入る入手先ヒント

検索クエリ例:
```
"<産地名> coffee farm story" site:perfectdailygrind.com OR site:sprudge.com
"<産地名> コーヒー 農家 ストーリー"
"<産地名> coffee origin altitude climate"
```

#### 🇯🇵 japan-scene（日本最新コーヒーシーン）【NEW】

「日本最新コーヒーシーン」カテゴリ記事の素材収集。定期的に更新が必要なドメイン。

調査項目:
- JBC（ジャパン バリスタ チャンピオンシップ）最新結果・優勝者・使用豆
- WBC（ワールド バリスタ チャンピオンシップ）日本代表の動向
- 注目の新規オープンスペシャルティカフェ（全国・首都圏）
- 注目の日本人ロースター・バリスタの動向
- コーヒーフェス・イベント情報（SCAJ・Munch等）
- 日本のコーヒーサブスクサービスの最新比較
- コンビニコーヒーの動向（セブン・ローソン・ファミマ）
- サードウェーブ以降のトレンド（第4の波・地域密着・サステナブル等）

検索クエリ例:
```
"JBC 2026" OR "ジャパンバリスタチャンピオンシップ 2026"
"スペシャルティコーヒー 新店 2026"
"コーヒー トレンド 2026 日本"
site:coffeemag.jp OR site:standartmag.com
```

保存先: `knowledge/japan-scene.md`（月次更新推奨）

#### 📍 kanagawa（神奈川コーヒーマップ）【NEW】

「神奈川コーヒーマップ」カテゴリ記事の素材収集。地元読者への具体的な情報源。

調査エリア:
- **横浜**: みなとみらい / 元町 / 中華街 / 関内 / 馬車道 / 野毛 / 白楽 / 綱島
- **鎌倉・湘南**: 北鎌倉 / 鎌倉駅周辺 / 江ノ島 / 茅ヶ崎 / 逗子 / 葉山
- **川崎**: 武蔵小杉 / 溝の口 / 川崎駅周辺
- **西エリア**: 小田原 / 箱根 / 平塚 / 藤沢 / 厚木

調査項目:
- エリア別のおすすめスペシャルティカフェ（直近1〜2年のオープン店を優先）
- 神奈川発の注目ロースター情報
- 神奈川のコーヒーイベント情報
- アクセス情報（駅からの距離・所要時間）
- カフェの雰囲気・特徴（週末散歩向き / 一人作業向き / デート向き 等）

検索クエリ例:

```
"神奈川 コーヒー ロースター"

```

保存先: `knowledge/kanagawa.md`（季節ごとの更新推奨）

### Step 2: 知識の構造化

収集した情報を以下のテンプレートに沿って Markdown ファイルに整理する。

**保存先:** `knowledge/<ドメイン>.md`

```markdown
# <ドメイン名> — コーヒー知識ベース

> 最終更新: <日付> | ソース数: <n>件

## 概要
（3〜5文のサマリー）

## 核心知識
（箇条書きで重要事項を列挙）

## 詳細セクション
（ドメイン別の詳細内容）

## フレーバー/比較マトリクス
（表形式で視覚化できる情報）

## 記事アイデア
（このドメインから派生する記事トピック候補 5〜10件）

## 参考情報
（調査で判明した信頼性の高い情報源）
```

```markdown
|--------|----------|----------|----------|
| ⭐⭐⭐ | ... | ... | ... |
```

### Step 4: インデックス更新

`knowledge/INDEX.md` に各ドメインファイルへのリンクと要約を記載する:

```markdown
| ...      | ...      | ...        | ...        | ...    |
```
