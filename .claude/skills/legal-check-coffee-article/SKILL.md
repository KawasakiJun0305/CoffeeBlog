# コーヒー記事 リーガルチェックスキル

note.com に公開するコーヒー記事の法的リスクを7つの観点でチェックする。
日本の主要法令（景品表示法・薬機法・著作権法・ステマ規制等）に基づき、
リスク箇所を特定・分類して修正提案を出力する。`--fix` 指定時は自動修正する。

## 使い方

```
/legal-check-coffee-article [ファイルパス] [--fix] [--strict]
```

**引数:**
- `[ファイルパス]`: チェック対象ファイル（省略時は `output/` 内の最新ファイルを使用）
- `--fix`: 問題箇所を自動修正してファイルを上書き保存
- `--strict`: 厳格モード（合格ラインを 70→85 に引き上げ。公開前最終確認用）

---

## チェック対象の根拠法令

| 法令 | 主なリスク |
|------|-----------|
| 景品表示法 | 誇大広告・不当表示 |
| 薬機法（医薬品医療機器等法） | 根拠なき健康効果の標榜 |
| 著作権法 | 無断転載・引用ルール違反 |
| 不正競争防止法 | 競合他社への虚偽事実の告知 |
| ステルスマーケティング規制（2023年〜） | PR・広告の非開示 |
| 個人情報保護法 | 個人特定につながる情報の掲載 |
| 食品表示法・健康増進法 | 食品への効能・効果の標榜 |

---

## 実行手順

### Step 0: ファイル特定

1. ファイルパスが指定されていれば Read ツールで読み込む
2. 省略時は `output/` を Glob で検索し、最新の `.md` ファイルを対象とする
3. frontmatter の `topic` / `category` / `date` を読み取り、チェック基準を決定する

---

### Step 1: 7項目チェック

各項目について **リスクレベル（🔴 高 / 🟡 中 / 🟢 低）** と **該当箇所の引用** を出力する。

---

#### Check 1: 健康・医療効果の過大表示 /20

根拠なき健康効果・医療的効能を標榜していないか。
薬機法・健康増進法違反につながるリスクがある。

チェック項目:
- [ ] 「コーヒーを飲むと〇〇が治る・予防できる」等の断定
- [ ] 「〇〇に効く」「〇〇に効果がある」という医薬品的表現
- [ ] 「ダイエット効果」「血糖値を下げる」等、根拠のない機能性表示
- [ ] 「病気」「症状」「治療」「予防」など医療用語との直接的な組み合わせ
- [ ] 「妊婦に良い / 悪い」等の医学的根拠を必要とする主張

リスク判定:
- 🔴 高: 医療的効能を断定している → 即時修正必須
- 🟡 中: 曖昧な健康言及（「体に良い」「元気になる」程度）→ 表現を弱める
- 🟢 低: 問題なし

修正方向:
- 断定 → 「〜という研究報告があります」「〜と言われています」
- 医療用語 → 「日常的な楽しみとして」「気分転換に」

---

#### Check 2: 広告・PR の非開示（ステマ規制） /15

2023年10月施行のステルスマーケティング規制に対応しているか。

チェック項目:
- [ ] カフェや商品を無償提供・招待されて書いた記事に「PR」「提供」表記があるか
- [ ] アフィリエイトリンクを含む場合に開示表記があるか
- [ ] 特定の店舗・ブランドを一方的に絶賛しており、対価を受け取っている可能性があるか
- [ ] note.com のルール上求められる広告開示がなされているか

リスク判定:
- 🔴 高: 対価を受け取っており開示なし → 冒頭に「※本記事はPRを含みます」等を追記
- 🟡 中: 開示が末尾のみ、または目立たない位置 → 冒頭への移動を推奨
- 🟢 低: 自費・自主執筆、開示不要

※判断できない場合は 🟡 中として注記する。

---

#### Check 3: 著作権・無断転載リスク /15

他者の著作物を適切に引用・利用しているか。

チェック項目:
- [ ] 他のブログ・メディア・SNSからの文章の無断コピーがないか
- [ ] 引用の場合、引用元の明示と「引用」の体裁（独立段落・出典明記）があるか
- [ ] 写真・イラストについて「[画像: 〇〇]」プレースホルダーの説明が自分で用意した素材か確認できるか
- [ ] 歌詞・書籍の一節などを長文引用していないか
- [ ] 他記事と酷似した構成・文体になっていないか（パクリ疑惑リスク）

リスク判定:
- 🔴 高: 明らかな無断引用・コピー
- 🟡 中: 引用元不明または引用体裁の不備
- 🟢 低: 問題なし

---

#### Check 4: 誹謗中傷・名誉毀損リスク /15

特定の個人・店舗・ブランドに対する不当な批判がないか。

チェック項目:
- [ ] 特定の店舗・ブランドへの根拠なき批判・否定的評価（「まずい」「詐欺だ」等）
- [ ] 実在する個人バリスタ・店主への侮辱的な表現
- [ ] 競合店・競合商品の虚偽事実に基づく否定
- [ ] カフェレビューで「私の経験ベース」の主観と断りなく事実のように記述
- [ ] 閉店・改名した店への古い情報のまま否定的な記述

リスク判定:
- 🔴 高: 根拠なき事実に基づく誹謗・中傷
- 🟡 中: 主観表現だが過激な表現が含まれる → 「個人的な感想」と明記
- 🟢 低: 問題なし

---

#### Check 5: 店舗・価格情報の正確性 /15

不正確な情報を断定している箇所がないか。

チェック項目:
- [ ] 価格（メニュー価格、豆の単価）を断定的に記載していないか → 「目安として」「〜円前後」「公式サイトでご確認ください」が必要
- [ ] 営業時間・定休日を断定的に記載していないか → 「公式SNS・サイトでご確認ください」が必要
- [ ] 店舗の住所・アクセスは正確か（架空の情報・古い情報ではないか）
- [ ] 「現在開催中のイベント」など時制依存の情報に注意書きがあるか
- [ ] 廃業・移転した店舗を現在形で紹介していないか

リスク判定:
- 🔴 高: 架空の価格・住所・営業時間を断定記載
- 🟡 中: 断定的な価格・営業時間記載（確認促進なし）→ 免責注記を追加
- 🟢 低: 「公式サイトでご確認ください」が適切に付いている

---

#### Check 6: 個人情報・プライバシーリスク /10

個人を特定しうる情報を不用意に掲載していないか。

チェック項目:
- [ ] 実名バリスタ・生産者への言及は公開情報に基づいているか
- [ ] 同行者・SNSの友人投稿など第三者のプライバシーを侵害していないか
- [ ] 店内での他の客の様子を個人特定できるように記述していないか
- [ ] 農家・生産者の収入・経営状況を根拠なく記述していないか

リスク判定:
- 🔴 高: 個人特定可能な情報の無断掲載
- 🟡 中: 氏名は伏せているが状況から特定可能
- 🟢 低: 問題なし

---

#### Check 7: 比較広告・根拠なき優位性表示 /10

根拠なき他社比較・優位性表示がないか。

チェック項目:
- [ ] 「業界最高」「日本一」「〇〇と比べて断然美味しい」等の根拠なき最上級表現
- [ ] 他社・他製品を名指しして根拠なく劣っていると示唆
- [ ] 抽出数値・品質評価で「最高のコーヒーはこれだ」と断定
- [ ] 読者に特定の購買行動を誘導する場合の根拠の妥当性

リスク判定:
- 🔴 高: 根拠なき最上級主張・名指し批判
- 🟡 中: 主観的優位性表示（「個人的に一番好き」等の明記なし）
- 🟢 低: 問題なし

---

```
║ Check 1 健康・医療効果の過大表示   : __/20       ║
║ Check 2 ステマ規制・PR非開示       : __/15       ║
║ Check 3 著作権・無断転載           : __/15       ║
║ Check 4 誹謗中傷・名誉毀損         : __/15       ║
║ Check 5 店舗・価格情報の正確性     : __/15       ║
║ Check 6 個人情報・プライバシー     : __/10       ║
║ Check 7 比較広告・優位性表示       : __/10       ║
╠══════════════════════════════════════════════════╣
║ 合計スコア                        : __/100       ║
║ 判定                              : [判定ラベル] ║
╚══════════════════════════════════════════════════╝
```

**判定基準（通常モード）:**

| スコア | 判定 | 対応 |
|--------|------|------|
| 90〜100 | ✅ 法的リスクなし | 公開可 |
| 75〜89  | ⚠️ 軽微なリスク | 🟡項目を修正してから公開 |
| 60〜74  | 🔧 要修正       | 🔴項目を修正し再チェックを推奨 |
| 0〜59   | ❌ 公開不可     | 即時修正必須。公開を保留 |

`--strict` 指定時は合格ラインが **75→85** に引き上げ。

### Step 3: フィードバック出力

```
## 🔴 高リスク（公開前に必ず修正）
<該当テキストを引用して指摘 + 修正案>

## 🟡 中リスク（修正を強く推奨）
<該当テキストを引用して指摘 + 修正案>

## 🟢 低リスク（問題なし）
<該当チェック項目のリスト>

## 📋 修正例
（「この文を〇〇に変える」という具体例を1〜3個）
```

---

### Step 4: 自動修正（--fix 指定時のみ）

以下を自動修正してファイルを上書き保存する:

**自動修正対象:**
- 断定的な健康効果表現 → 「〜という研究があります」に軟化
- 価格・営業時間の断定記載 → 末尾に「※公式サイトでご確認ください」を追記
- 「最高」「日本一」等の根拠なき最上級表現 → 「個人的に気に入っている」等に修正
- 引用表記の不備 → 引用符と出典フォーマットを整形

**自動修正しないもの（人間の判断が必要）:**
- ステマ規制（PR開示要否は執筆者本人のみ判断可能）
- 誹謗中傷・名誉毀損（削除か存続かの判断が必要）
- 著作権侵害の疑い（引用か転載かの判断が必要）

修正後に差分サマリーを出力する。

## カテゴリ別注意事項

| カテゴリ | 特に注意すべきリスク |
|---------|-------------------|
| 豆・産地 | 健康効果の標榜（Check 1）・産地情報の根拠（Check 5） |
| 抽出・レシピ | 健康強調（Check 1）・数値の正確性（Check 5） |
| カフェ情報 | 価格・営業時間（Check 5）・誹謗中傷（Check 4）・ステマ（Check 2） |
| 道具・機器 | 比較広告（Check 7）・価格断定（Check 5）・ステマ（Check 2） |
| 日本最新コーヒーシーン | 著作権（Check 3）・個人情報（Check 6）・事実正確性（Check 5） |
| 神奈川コーヒーマップ | 店舗情報の鮮度（Check 5）・ステマ（Check 2） |
| コーヒー文化 | 著作権（Check 3）・健康言及（Check 1） |
